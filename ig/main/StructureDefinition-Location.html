Logical Model: Location OMOP Table

Official URL: https://health-data-hub.fr/ig/fhir/eds/StructureDefinition/Location Version: 0.1.0
Draft as of 2024-04-12 Computable Name: Location

The LOCATION table represents a generic way to capture physical location or address information of Persons and Care Sites.

Usage:

  • Refer to this Logical Model Profile: CareSite OMOP Table and Person OMOP Table
