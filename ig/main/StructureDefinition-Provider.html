Logical Model: Provider OMOP Table

Official URL: https://health-data-hub.fr/ig/fhir/eds/StructureDefinition/Provider Version: 0.1.0
Draft as of 2024-04-12 Computable Name: Provider

The PROVIDER table contains a list of uniquely identified healthcare providers. These are individuals providing hands-on healthcare to patients, such as physicians, nurses, midwives, physical therapists etc.

Usage:

  • Refer to this Logical Model Profile: Condition Occurrence OMOP Table, Device Exposure OMOP Table, Drug Occurrence OMOP Table, Measurement OMOP Table...
